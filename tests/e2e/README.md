# Flow UI Playwright Suite

## What it validates

- Landing clarity / mental map cues
- Semantic drill interactions (double-click in/out)
- Business vs Engineering lens consistency
- Search and detail-panel accessibility
- Screenshot capture for visual review artifacts

## Run

```bash
npm run test:e2e
```

## Continuous visual loop

```bash
npm run test:e2e:loop
```

## Artifacts

- Playwright report: `playwright-report/`
- Failure assets: `test-results/`
- Golden review screenshots: `workflow/artifacts/ITER-001/qa/`
